# Mac (Apple Silicon) — INFERENCE ONLY.

# System dependency (install separately, not via pip):
#   LaTeX — `latex` + `dvipng`, needed for math rendering by
#   infer.py:_render_latex_native and tools/slide_editor.py.
#   Install BasicTeX, then:  sudo tlmgr update --self &&
#   sudo tlmgr install amsmath amsfonts dvipng standalone preview
#
#   LibreOffice — needed ONLY to accept PowerPoint (.ppt/.pptx) input in the
#   slide editor (rasterised to PDF via headless `soffice`). PDF and image-
#   folder input do not need it.  Install:  brew install --cask libreoffice
#
# Install:  pip install -r requirements-mac.txt
# After a working test run in Germany, freeze exact versions:
#   pip freeze > requirements-mac.lock.txt   (then prefer the lock file)
